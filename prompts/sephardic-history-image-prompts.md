## Platform Versions

### TikTok — 9:16 vertical (1080 × 1920)

Append to each prompt:
```
vertical composition, subject centered in frame, portrait orientation, 9:16 aspect ratio,
negative space above and below for text overlay
```

**Midjourney:** add `--ar 9:16 --v 6.1` at the end
**DALL-E 3:** set size to `1024x1792` in the API or select portrait in the UI
**Adobe Firefly:** set canvas to 1080×1920 before generating

---

### YouTube — 16:9 horizontal (1920 × 1080)

Append to each prompt:
```
wide cinematic composition, full scene visible, landscape orientation, 16:9 aspect ratio,
room for text overlay at bottom third
```

**Midjourney:** add `--ar 16:9 --v 6.1` at the end
**DALL-E 3:** set size to `1792x1024` in the API or select landscape in the UI
**Adobe Firefly:** set canvas to 1920×1080 before generating
